# Česky

## 1. Rozšíření Synchronizace (Desktop App)
Aktuálně podporujeme **Librera Reader** přes USB.
- **[PLÁNOVÁNO] Podpora dalších aplikací**:
  - **Moon+ Reader**: Synchronizace souborů `.po` / `.mrpro`.
  - **ReadEra**: Parsování záloh `library.json`.
- **[PLÁNOVÁNO] Bezdrátová synchronizace**:
  - Odstranění nutnosti USB kabelu.
  - Využití Wireless ADB nebo přímého síťového přenosu.

## 2. Komunitní a Sociální Funkce (Web)
- **[PLÁNOVÁNO] Import z Goodreads/StoryGraph**:
  - Možnost stáhnout hodnocení a recenze z externích služeb.
- **[PLÁNOVÁNO] Interní Recenze**:
  - Propojení komentářů u knih přímo s fórem (diskuze o knize).
- **[PLÁNOVÁNO] Gamifikace**:
  - Odznaky za přečtené série.
  - Čtenářské výzvy (např. "Přečti 10 knih v tomto roce").

## 3. Vylepšení Audirotra
- **[PLÁNOVÁNO] Auto-Fix**:
  - Možnost, aby Auditor sám opravil drobné chyby (např. doplnil chybějící jazyk podle metadat).

# English

## 1. Sync Ecosystem Expansion (Desktop App)
Currently supporting **Librera Reader** via USB.
- **[PLANNED] Multi-App Support**:
  - **Moon+ Reader**: Syncing `.po` / `.mrpro` files.
  - **ReadEra**: Parsing `library.json` backups.
- **[PLANNED] Wireless Sync**:
  - Removing the need for USB cables.
  - Utilizing Wireless ADB or direct network transfer.

## 2. Community & Social Features (Web)
- **[PLANNED] Goodreads/StoryGraph Import**:
  - Pulling ratings and reviews from external services.
- **[PLANNED] Internal Reviews**:
  - Linking book comments directly to forum threads.
- **[PLANNED] Gamification**:
  - Badges for completed series.
  - Reading challenges (e.g., "Read 10 books this year").

## 3. Auditor Improvements
- **[PLANNED] Auto-Fix**:
  - Allowing the Auditor to automatically repair minor issues (e.g., fill missing languages from metadata).
