# Česky

## Hlášení bezpečnostních chyb
Pokud objevíte bezpečnostní zranitelnost (např. možnost obejít přihlášení, SQL injection, XSS), prosím **nezveřejňujte ji ihned veřejně**.

1. **Kontakt**: Napište soukromou zprávu (PM) administrátorovi `Ariael` na fóru **ebookforum.sk**.
2. **Reakce**: Pokusíme se chybu ověřit a opravit v co nejkratším čase (obvykle do 48 hodin).

## Podporované verze
Bezpečnostní záplaty jsou aplikovány pouze na **nejnovější verzi** v repozitáři `main`. Doporučujeme pravidelně aktualizovat pomocí skriptu `cwupdate`.

---

# English

## Reporting a Vulnerability
If you discover a security vulnerability, please **do not disclose it publically** immediately.

1. **Contact**: Send a Private Message (PM) to admin `Ariael` on **ebookforum.sk**.
2. **Response**: We will aim to verify and patch the issue ASAP (usually within 48 hours).

## Supported Versions
Security patches are applied only to the **latest version** on the `main` branch. We recommend regular updates using the `cwupdate` script.
